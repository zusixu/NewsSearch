# reporting-output

## 做什么

生成每日 Top 10 Markdown/JSON 报告，沉淀证据、逻辑链、A 股关注点、风险与历史归档。

## 怎么做

1. 定义日报输出数据结构。
2. 设计 Markdown 模板。
3. 设计 JSON 结构。
4. 输出历史归档目录。
5. 保留与 prompt/profile/运行日志的关联信息。

已确认输出要求：
- 日报中 A 股映射应覆盖三层：行业/板块、候选标的池、具体个股；
- 所有结论必须附带强弱分级；
- 同一天可能存在多批次报告，报告命名与归档必须能区分“开盘前批次”和“午间批次”。

## 完成定义

- 每天可生成可读的 Markdown 日报。
- 同步生成可供后续程序和 AI 复用的 JSON。
- 报告中能看到证据、A 股映射、风险和排序理由。

## 依赖

- `llm-analysis`
- `a-share-mapping`
