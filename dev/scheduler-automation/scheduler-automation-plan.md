# scheduler-automation

## 做什么

实现本地每日自动运行、补跑、日志落盘、失败重试和 Windows Task Scheduler 集成。

## 怎么做

1. 建立 PowerShell 运行脚本。
2. 建立计划任务注册脚本。
3. 支持每日固定时间执行。
4. 支持补跑指定日期。
5. 支持失败日志和重试策略。

已确认调度要求：
- 默认每天运行两次：开盘前一小时一次、下午两点一次。
- 调度实现应支持同一自然日生成多个批次结果。

## 完成定义

- 你可以一键注册每日任务。
- 任务可在本地无人值守执行。
- 失败后可查看日志并手动补跑。
